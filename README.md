# Flask-MySQL-CRUD
This is a tool created with Flask, SQLAlchemy, HTML, Bootstrap 4 and MySQL.

It is a simple contact book.

## To run the program on your computer write the following in the shell

```
pip install -r requirements.txt
```
After:

```
python index.py
```
